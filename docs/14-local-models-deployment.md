# 本地模型部署方案

> 基于32个固定数字员工需求，规划最优本地模型部署配置

## 一、三层LLM架构

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                    三层LLM架构 - 核心模型                                      │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  Layer 1: 主大脑 (MainBrain) - Qwen3.5-27B                                  │
│  ├── 职责: 复杂推理、跨部门协调、战略决策、多模态处理                          │
│  ├── VRAM需求: 20-24GB (量化) / 54GB (FP16)                                 │
│  ├── 适用员工: M01协调员、M02战略规划师、跨部门任务                            │
│  └── 推理速度: 30-50 tokens/s (RTX 4090)                                    │
│                                                                             │
│  Layer 2: 闲聊神经元 (Qwen3Neuron) - Qwen3-0.6B                             │
│  ├── 职责: 日常对话、快速响应、简单任务、离职/外来人员闲聊                     │
│  ├── VRAM需求: 1-2GB                                                        │
│  ├── 适用员工: 所有部门的日常沟通                                             │
│  └── 推理速度: 100-150 tokens/s                                              │
│                                                                             │
│  Layer 3: 工具神经元 (ToolNeuron) - Qwen3.5-2B (默认) / BitNet-1.58-3B (备选)     │
│  ├── 职责: 工具检测、兜底处理、触发进化信号、动态模型选择                          │
│  ├── VRAM需求: 4GB (Qwen3.5-2B) / 1GB (BitNet)                              │
│  ├── 上下文: 262K (Qwen3.5-2B) / 4K (BitNet)                                 │
│  ├── 多模态: ✅ 原生支持 (Qwen3.5-2B)                                         │
```

## 四、Layer 3 工具神经元模型选择策略

### 4.1 模型对比

| 指标 | Qwen3.5-2B (默认) | BitNet-1.58-3B (备选) |
|------|------------------|---------------------|
| **MMLU-Pro** | 66.5 | ~45 |
| **BFCL-V4 (工具调用)** | 43.6 | ~25 |
| **TAU2-Bench (Agent)** | 48.8 | ~20 |
| **上下文长度** | 262K | 4K |
| **多模态** | ✅ 原生支持 | ❌ |
| **内存需求** | 4GB | 1GB |
| **推理速度** | 60-100 tokens/s | 80-120 tokens/s |
| **推荐场景** | GPU推理、高性能 | CPU推理、边缘设备 |

### 4.2 自动切换逻辑

```java
// ToolNeuronModelSelector 自动选择逻辑
if (availableMemoryMB >= 4096 && availableProcessors >= 4 && cpuLoad < 0.8) {
    return ModelType.QWEN35_2B;  // 默认推荐
} else {
    return ModelType.BITNET_158_3B;  // 资源受限时降级
}
```

### 4.3 配置示例

```yaml
# application.yml
tool-neuron:
  model:
    default: qwen3.5-2b        # 默认模型
    auto-select: true          # 启用自动选择
    memory-threshold-mb: 2048  # 内存阈值
    cpu-load-threshold: 0.8    # CPU负载阈值
```

### 4.4 部署建议

| 场景 | 推荐模型 | 说明 |
|------|---------|------|
| **服务器 (GPU)** | Qwen3.5-2B | 高性能、多模态、长上下文 |
| **服务器 (CPU)** | BitNet-1.58-3B | 低内存、快速响应 |
| **边缘设备** | BitNet-1.58-3B | 资源受限场景 |
| **嵌入式设备** | BitNet-1.58-3B | 最小资源需求 |
