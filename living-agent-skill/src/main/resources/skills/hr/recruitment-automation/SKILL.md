# Recruitment Automation

> 招聘自动化技能，自动执行招聘流程、简历筛选、面试安排

## 一、技能描述

招聘自动化技能使人力资源部数字员工能够自动执行招聘相关任务，包括：
- 职位发布自动化
- 简历筛选和评分
- 面试安排和提醒
- 候选人沟通自动化

## 二、适用编制

| 编制代码 | 职位 | 主要用途 |
|---------|------|---------|
| H01 | 招聘专员 | 简历筛选、面试安排 |
| H02 | 绩效管理员 | 入职流程管理 |

## 三、触发词

- 招聘、recruitment、招聘管理
- 简历筛选、候选人筛选
- 面试安排、面试邀请
- 职位发布、job posting

## 四、功能模块

### 4.1 职位管理

| 功能 | 说明 |
|------|------|
| 职位创建 | 创建职位描述、要求、薪资范围 |
| 多平台发布 | 同步到招聘网站（智联、前程无忧、BOSS直聘） |
| 职位状态管理 | 开放/暂停/关闭 |

### 4.2 简历筛选

| 功能 | 说明 |
|------|------|
| 自动解析 | 解析PDF/Word格式简历 |
| 关键词匹配 | 根据职位要求匹配关键词 |
| 智能评分 | AI评分候选人匹配度 |
| 自动标签 | 标记技能、经验、学历 |

### 4.3 面试管理

| 功能 | 说明 |
|------|------|
| 日程协调 | 自动协调面试官和候选人时间 |
| 邀请发送 | 发送面试邀请邮件/短信 |
| 提醒通知 | 面试前1天/1小时提醒 |
| 反馈收集 | 面试后收集面试官评价 |

## 五、使用示例

### 5.1 发布职位

```
用户: 发布一个Java后端工程师职位

技能执行:
1. 创建职位信息模板
2. 确认职位要求：3-5年经验、Spring Boot、微服务
3. 同步到BOSS直聘、智联招聘
4. 返回职位链接
```

### 5.2 筛选简历

```
用户: 筛选Java工程师职位的简历

技能执行:
1. 获取待处理简历列表
2. 解析简历内容
3. 匹配职位关键词
4. 生成评分报告
5. 推荐Top 10候选人
```

### 5.3 安排面试

```
用户: 为候选人张三安排技术面试

技能执行:
1. 查询面试官可用时间
2. 发送时间选项给候选人
3. 确认面试时间
4. 发送面试邀请
5. 创建日历事件
```

## 七、自动化规则

| 触发条件 | 自动动作 |
|---------|---------|
| 新简历投递 | 自动解析、评分、标签 |
| 评分>80 | 自动发送初筛通过邮件 |
| 面试确认 | 创建日历事件、发送提醒 |
| 面试后24h | 发送评价表给面试官 |
| Offer接受 | 启动入职流程 |

## 八、集成平台

| 平台 | 功能 | API支持 |
|------|------|--------|
| BOSS直聘 | 职位发布、简历获取 | ✅ |
| 智联招聘 | 职位发布、简历获取 | ✅ |
| 前程无忧 | 职位发布、简历获取 | ✅ |
| 猎聘 | 高端职位 | ✅ |
| 钉钉 | 面试安排、通知 | ✅ |

## 九、配置要求

```yaml
recruitment:
  platforms:
    boss:
      app_key: ${BOSS_APP_KEY}
      app_secret: ${BOSS_APP_SECRET}
    zhilian:
      api_key: ${ZHILIAN_API_KEY}
  
  scoring:
    skill_weight: 0.4
    experience_weight: 0.3
    education_weight: 0.2
    stability_weight: 0.1
  
  notification:
    email_provider: sendgrid
    sms_provider: aliyun
```
