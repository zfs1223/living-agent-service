# Budget Management

> 预算管理技能，月度预算管理、超支预警、预算分析

## 一、技能描述

预算管理技能使财务部数字员工能够管理企业预算相关任务，包括：
- 月度/季度/年度预算设定
- 实时预算监控
- 超支预警通知
- 预算执行分析

## 二、适用编制

| 编制代码 | 职位 | 主要用途 |
|---------|------|---------|
| F04 | 预算管理员 | 预算设定、监控、分析 |
| F03 | 成本核算员 | 成本预算分析 |
| F01 | 财务会计 | 预算执行跟踪 |

## 三、触发词

- 预算、budget、预算管理
- 超支、预算预警

## 五、使用示例

### 5.1 创建预算

```
用户: 为技术部创建Q2预算

技能执行:
1. 获取历史预算数据
2. 分析支出趋势
3. 生成预算建议
4. 创建预算审批流程
5. 返回预算ID
```

### 5.2 预算监控

```
用户: 查看本月预算执行情况

技能执行:
1. 汇总各部门预算使用
2. 计算使用率
3. 识别超支风险
4. 生成监控报告
```

### 5.3 超支预警

```
用户: 检查预算超支风险

技能执行:
1. 分析当前支出速度
2. 预测月底超支风险
3. 生成预警列表
4. 发送通知给相关部门
```

## 七、预警规则

| 使用率 | 预警级别 | 动作 |
|--------|---------|------|
| < 50% | 正常 | 无 |
| 50-70% | 关注 | 记录日志 |
| 70-85% | 警告 | 发送邮件通知 |
| 85-100% | 严重 | 发送多渠道通知 |
| > 100% | 超支 | 升级主管审批 |

## 八、自动化规则

| 触发条件 | 自动动作 |
|---------|---------|
| 月初 | 发送预算概览给部门负责人 |
| 使用率>70% | 发送预警邮件 |
| 使用率>85% | 发送钉钉/企微通知 |
| 月末 | 生成预算执行报告 |
| 超支 | 冻结支出，需审批 |

```yaml
    recipients:
      - finance@company.com
      - ${dept_manager_email}
  
  approval:
    overspend:
      threshold: 10000
      approvers: [cfo, ceo]
```
