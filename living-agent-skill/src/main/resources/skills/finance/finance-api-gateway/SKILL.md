# Finance API Gateway

> 财务系统API网关，连接财务软件、银行接口、税务系统

## 一、技能描述

财务API网关技能使数字员工能够与各类财务系统进行集成操作，包括：
- ERP系统对接（金蝶、用友、SAP）
- 银行接口对接（查询余额、交易流水）
- 税务系统对接（发票查验、纳税申报）
- 支付平台对接（支付宝、微信支付）

## 二、适用编制

## 五、使用示例

### 5.1 发票查验

```
用户: 帮我查验这张发票真伪
发票代码: 1100...
发票号码: 12345...

技能执行:
1. 调用国家税务总局发票查验API
2. 返回发票信息：金额、开票方、商品明细
3. 验证发票状态：有效/作废/红冲
```

### 5.2 银行余额查询

```
用户: 查询公司招商银行账户余额

技能执行:
1. 调用招商银行企业API
2. 返回账户余额和可用余额
3. 显示最近5笔交易
```

### 5.3 ERP凭证录入

```
用户: 在金蝶系统中录入一笔费用凭证

技能执行:
1. 连接金蝶云API
2. 创建凭证：借-管理费用，贷-银行存款
3. 返回凭证号
```

## 六、配置要求

```yaml
finance:
  erp:
    type: kingdee  # kingdee/yonyou/sap
    api_url: ${ERP_API_URL}
    app_key: ${ERP_APP_KEY}
    app_secret: ${ERP_APP_SECRET}
  
  bank:
    cmb:
      api_url: https://api.cmbchina.com
      client_id: ${CMB_CLIENT_ID}
      client_secret: ${CMB_CLIENT_SECRET}
  
  tax:
    api_url: https://inv-veri.chinatax.gov.cn
    app_key: ${TAX_APP_KEY}
```

## 七、安全要求

1. 所有API调用必须使用HTTPS
2. 敏感凭证存储在加密配置中
3. 操作日志完整记录
4. 大额操作需要审批

## 八、错误处理

| 错误类型 | 处理方式 |
|---------|---------|
| API超时 | 重试3次，间隔递增 |
| 认证失败 | 刷新Token后重试 |
| 余额不足 | 返回错误，通知管理员 |
| 发票查验失败 | 返回错误信息，建议人工核实 |
