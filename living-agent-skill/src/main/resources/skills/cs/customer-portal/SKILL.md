# Customer Portal

> 客户门户技能，管理客户自助服务、知识库、FAQ

## 一、技能描述

客户门户技能使客服部数字员工能够管理客户自助服务平台，包括：
- 知识库文章管理
- FAQ自动问答
- 客户自助服务
- 智能客服机器人

## 二、适用编制

| 编制代码 | 职位 | 主要用途 |
|---------|------|---------|
| C01 | 客服专员 | 知识库维护、FAQ管理 |
| C02 | 工单处理员 | 自助服务配置 |

## 三、触发词

- 知识库、knowledge base、KB
- FAQ、常见问题
- 自助服务、self-service
- 智能客服、聊天机器人

## 四、功能模块

### 4.1 知识库管理

| 功能 | 说明 |
|------|------|
| 文章创建 | 创建帮助文档 |
| 分类管理 | 按产品/功能分类 |
| 版本控制 | 文章版本管理 |
| 搜索优化 | 关键词、标签优化 |

### 4.2 FAQ管理

| 功能 | 说明 |
|------|------|
| 问题收集 | 从工单提取常见问题 |
| 答案编写 | 编写标准答案 |
| 自动推荐 | 根据问题推荐FAQ |
| 效果统计 | FAQ使用率统计 |

### 4.3 智能客服

| 功能 | 说明 |
|------|------|
| 意图识别 | 识别用户问题意图 |
| 答案匹配 | 匹配知识库答案 |
| 多轮对话 | 复杂问题引导 |
| 人工转接 | 无法解决时转人工 |

## 五、使用示例

### 5.1 创建知识库文章

```
用户: 创建一篇关于"如何重置密码"的知识库文章

技能执行:
1. 收集相关信息
2. 生成文章内容
3. 添加截图说明
4. 设置分类和标签
5. 发布到知识库
```

### 5.2 更新FAQ

```
用户: 根据最近工单更新FAQ

技能执行:
1. 分析最近100个工单
2. 提取高频问题
3. 生成FAQ条目
4. 关联知识库文章
5. 发布更新
```

### 5.3 智能客服配置

```
用户: 配置智能客服的欢迎语和常见问题

技能执行:
1. 设置欢迎语模板
2. 配置快捷回复
3. 设置意图识别规则
4. 配置转人工条件
5. 测试对话流程
```

## 七、FAQ模板

```markdown
# Q: 如何重置密码？

**分类**: 账户管理 > 安全设置
**适用版本**: 全部
**最后更新**: 2026-03-13

## 答案

您可以通过以下步骤重置密码：

1. 访问登录页面
2. 点击"忘记密码"
3. 输入注册邮箱
4. 查收重置邮件
5. 点击链接设置新密码

## 相关文章
- [如何设置强密码](/kb/security/strong-password)
- [账户安全最佳实践](/kb/security/best-practices)

## 标签
密码, 重置, 登录, 安全
```

## 九、统计指标

| 指标 | 说明 | 目标值 |
|------|------|--------|
| 自助解决率 | 无需人工干预解决 | > 60% |
| FAQ命中率 | 问题匹配到FAQ | > 80% |
| 知识库使用率 | 用户查看知识库 | > 40% |
| 平均对话轮数 | 解决问题对话次数 | < 3轮 |
| 转人工率 | 需要转人工 | < 20% |

## 十、配置要求

```yaml
customer_portal:
  knowledge_base:
    platform: zendesk  # zendesk/freshdesk/custom
    search_enabled: true
    rating_enabled: true
  
  faq:
    auto_extract: true
    update_frequency: weekly
  
  chatbot:
    enabled: true
    provider: custom  # custom/dialogflow/watson
    
    welcome_message: |
      您好！我是智能客服小助手。
      请问有什么可以帮您的？
    
    quick_replies:
      - "如何重置密码"
      - "如何升级套餐"
      - "联系人工客服"
    
    transfer_conditions:
      - sentiment: negative
      - confidence: "<0.5"
      - keyword: "人工"
      - keyword: "投诉"
```
