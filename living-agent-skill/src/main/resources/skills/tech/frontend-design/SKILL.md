# Frontend Design

> 前端界面设计技能，创建高质量、独特的前端界面

## 一、技能描述

前端设计技能使技术部和行政部数字员工能够创建专业级前端界面，包括：
- Web组件和页面设计
- React/Vue组件开发
- 响应式布局设计
- 视觉美学优化

## 四、设计原则

### 4.1 避免AI通用美学

❌ 避免使用：
- 过度使用的字体：Inter, Roboto, Arial
- 俗套配色：紫色渐变配白色背景
- 千篇一律的布局和组件模式
- 缺乏特色的通用设计

✅ 应该追求：
- 独特的字体选择和搭配
- 大胆的色彩方案
- 意想不到的布局
- 有记忆点的设计

### 4.2 设计维度

| 维度 | 说明 |
|------|------|
| **排版** | 选择独特、有特色的字体，避免通用字体 |
| **色彩** | 大胆的色彩方案，主色调配鲜明对比 |
| **动效** | CSS动画、微交互、页面加载效果 |
| **空间** | 非对称布局、重叠、对角线流动 |
| **背景** | 渐变网格、噪点纹理、几何图案 |

## 五、使用示例

### 5.1 创建落地页

```
用户: 创建一个产品落地页

技能执行:
1. 理解产品定位和目标用户
2. 选择大胆的美学方向
3. 设计独特的视觉元素
4. 实现响应式布局
5. 添加动效和交互
```

### 5.2 设计React组件

```
用户: 创建一个数据展示组件

技能执行:
1. 分析数据展示需求
2. 设计组件结构
3. 选择合适的可视化方式
4. 实现交互功能
5. 优化性能和可访问性
```

## 六、美学方向参考

| 风格 | 特点 | 适用场景 |
|------|------|---------|
| 极简主义 | 精简、克制、注重细节 | 高端产品、设计工作室 |
| 最大主义 | 复杂、丰富、视觉冲击 | 创意机构、艺术项目 |
| 复古未来 | 怀旧与科技感结合 | 科技产品、游戏 |
| 有机自然 | 自然元素、柔和曲线 | 健康产品、环保品牌 |
| 奢华精致 | 高级感、精致细节 | 奢侈品、高端服务 |
| 顽皮童趣 | 可爱、有趣、互动 | 儿童产品、娱乐品牌 |
