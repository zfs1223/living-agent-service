# Sales Automation

> 销售自动化技能，自动执行销售流程、跟进提醒、数据同步

## 一、技能描述

销售自动化技能使销售部数字员工能够自动执行重复性销售任务，包括：
- 跟进提醒自动化
- 销售邮件自动化
- 客户数据同步
- 销售报表自动生成

## 二、适用编制

| 编制代码 | 职位 | 主要用途 |
|---------|------|---------|
| S01 | 销售代表 | 跟进提醒、邮件发送 |
| S02 | 市场专员 | 营销活动自动化 |
| S03 | 渠道经理 | 平台数据同步 |

## 三、触发词

- 销售自动化、自动跟进
- 跟进提醒、客户提醒
- 自动邮件、销售流程
- 数据同步、报表生成

## 四、自动化场景

### 4.1 跟进提醒

| 场景 | 触发条件 | 动作 |
|------|---------|------|
| 新线索 | 线索创建后 | 1小时内首次联系提醒 |
| 商机推进 | 阶段变更后 | 24小时内跟进提醒 |
| 长期未联系 | 7天无互动 | 跟进提醒+主管通知 |
| 合同到期 | 到期前30天 | 续约跟进提醒 |

### 4.2 邮件自动化

| 场景 | 触发条件 | 动作 |
|------|---------|------|
| 欢迎邮件 | 新客户创建 | 发送公司介绍+产品手册 |
| 跟进邮件 | 会议后24小时 | 发送会议纪要+下一步计划 |
| 节日问候 | 法定节假日 | 发送节日祝福 |
| 生日祝福 | 客户生日 | 发送生日祝福+优惠券 |

### 4.3 数据同步

| 场景 | 触发条件 | 动作 |
|------|---------|------|
| 线索同步 | 新线索产生 | 同步到CRM |
| 成交同步 | 商机成交 | 同步到财务系统 |
| 客户更新 | 信息变更 | 同步到所有系统 |

## 五、使用示例

### 5.1 设置跟进提醒

```
用户: 为客户ABC科技设置明天下午的跟进提醒

技能执行:
1. 创建跟进任务
2. 设置提醒时间: 明天14:00
3. 发送日历邀请
4. 返回任务ID
```

### 5.2 自动发送跟进邮件

```
用户: 给上周拜访的客户发送跟进邮件

技能执行:
1. 查询上周拜访记录
2. 生成个性化邮件内容
3. 发送邮件
4. 记录发送日志
```

### 5.3 生成销售周报

```
用户: 生成本周销售周报

技能执行:
1. 汇总本周新增线索
2. 统计商机进展
3. 计算成交金额
4. 生成周报并发送给主管
```

## 六、配置要求

```yaml
sales_automation:
  follow_up:
    default_reminder_hours: 24
    escalation_days: 7
  
  email:
    provider: sendgrid  # sendgrid/mailchimp/aliyun
    templates:
      welcome: "d-xxx"
      follow_up: "d-yyy"
      birthday: "d-zzz"
  
  sync:
    crm_interval: 300  # 5分钟
    finance_interval: 3600  # 1小时
```

## 七、工作流模板

### 7.1 新客户跟进流程

```
新客户创建
    │
    ├── 立即: 分配销售代表
    │
    ├── 1小时后: 首次联系提醒
    │
    ├── 24小时后: 检查是否已联系
    │       └── 未联系 → 升级主管
    │
    └── 7天后: 检查跟进状态
            └── 无跟进 → 自动发送关怀邮件
```

### 7.2 商机成交流程

```
商机标记成交
    │
    ├── 立即: 创建合同草稿
    │
    ├── 通知: 财务部成本核算员
    │
    ├── 同步: 更新CRM状态
    │
    └── 发送: 成交感谢邮件
```
