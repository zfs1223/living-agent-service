# Contract Management

> 合同管理技能，管理合同生命周期、审批流程、到期提醒

## 一、技能描述

合同管理技能使法务部数字员工能够管理企业合同全生命周期，包括：
- 合同创建和模板管理
- 审批流程管理
- 合同归档和检索
- 到期提醒和续签

## 二、适用编制

| 编制代码 | 职位 | 主要用途 |
|---------|------|---------|
| L01 | 合同审查员 | 合同审查、风险评估 |
| L02 | 合规专员 | 合规检查、政策审核 |

## 五、使用示例

### 5.1 创建合同

```
用户: 创建一份销售合同
甲方: ABC科技有限公司
乙方: XYZ贸易公司
金额: 50万元

技能执行:
1. 选择销售合同模板
2. 填充合同变量
3. 生成合同草稿
4. 发起审批流程
5. 返回合同编号
```

### 5.2 合同审查

```
用户: 审查合同 CTR-2026-001

技能执行:
1. 解析合同条款
2. 检查风险条款
3. 对比历史合同
4. 生成审查报告
5. 提出修改建议
```

### 5.3 到期提醒

```
用户: 查看下月到期的合同

技能执行:
1. 查询合同到期列表
2. 筛选下月到期合同
3. 检查续签状态
4. 生成提醒列表
5. 发送续签提醒
```

## 六、合同类型

| 类型 | 模板 | 审批流程 |
|------|------|---------|
| 销售合同 | sales_contract | 销售→法务→财务→总经理 |
| 采购合同 | purchase_contract | 采购→法务→财务→总经理 |
| 劳动合同 | employment_contract | HR→法务→总经理 |
| 保密协议 | nda | 法务→总经理 |
| 服务协议 | service_agreement | 业务→法务→财务 |

## 八、风险检查项

| 检查项 | 说明 | 风险等级 |
|--------|------|---------|
| 主体资格 | 签约主体是否合法 | 高 |
| 违约责任 | 违约条款是否对等 | 中 |
| 付款条款 | 付款条件是否合理 | 中 |
| 知识产权 | IP归属是否明确 | 高 |
| 保密条款 | 保密义务是否完整 | 中 |
| 争议解决 | 管辖条款是否合理 | 低 |
| 有效期限 | 合同期限是否明确 | 低 |

## 九、自动化规则

| 触发条件 | 自动动作 |
|---------|---------|
| 合同创建 | 发起审批流程 |
| 审批超时 | 发送催办提醒 |
| 合同签署 | 自动归档 |
| 到期前30天 | 发送续签提醒 |
| 到期前7天 | 升级提醒 |

## 十、配置要求

```yaml
contract:
  storage:
    type: minio  # minio/oss/local
    endpoint: ${MINIO_ENDPOINT}
    bucket: contracts
  
  approval:
    default_timeout: 72h
    reminder_interval: 24h
  
  templates:
    path: /templates/contracts
    variables:
      - company_name
      - party_name
      - amount
      - start_date
      - end_date
  
  reminder:
    first_reminder: 30d
    second_reminder: 14d
    final_reminder: 7d
  
  esign:
    provider: fadada  # fadada/esign
    app_id: ${ESIGN_APP_ID}
    app_secret: ${ESIGN_APP_SECRET}
```
